# local-ai
Local, privacy-first AI automation stack with n8n, Ollama, Qdrant, Whisper, Kokoro-TTS, MinIO, Pandoc, and Gotenberg. Build offline LLM agents, RAG pipelines, speech workflows, and document automation. Runs via Docker (CPU/GPU) on Windows WSL or macOS. Ideal for research, simulation, and secure data workflows.
